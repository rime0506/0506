# 0506

## 最新更新 (2026-02-05)

### ✨ 新增功能

#### 1. 聊天总结可编辑
- ✅ 在聊天总结详情页添加「编辑」按钮
- ✅ 用户可以手动修改AI自动生成的总结内容
- ✅ 支持保存/取消编辑操作
- ✅ 编辑后会更新时间戳

使用方法：
1. 进入「收藏」页面，选择「聊天总结」分类
2. 点击任意总结条目查看详情
3. 点击右上角「编辑」按钮进入编辑模式
4. 修改内容后点击「保存」或「取消」

#### 2. NPC角色完全可编辑
- ✅ 通过AI生成的NPC可以在「角色档案 > NPC」标签页中查看
- ✅ 点击任意NPC卡片即可编辑所有信息
- ✅ 支持修改：名字、昵称、人设描述、头像、虚拟身份信息等
- ✅ 鼠标悬停卡片时显示编辑图标提示
- ✅ 编辑NPC时会显示友好提示

使用方法：
1. 在「角色档案」中使用「批量生成NPC」功能
2. 生成后切换到「NPC」标签页
3. 点击任意NPC卡片进入编辑页面
4. 修改完成后点击「保存」

### 🎨 UI/UX 优化
- 角色卡片添加悬停效果和编辑图标
- NPC编辑页面添加AI生成提示
- 生成成功后提示用户可以编辑NPC
